# 神圣珍珠

<ItemImage id="irons_spellbooks:divine_pearl" scale="3" />

此物品可作为Iron珠宝的<Color id="light_purple">宝石</Color>。

### 属性
- 增加Iron法术的施法移动速度（百分比）

### 免疫效果
- 破甲

### 正面效果
- 神圣守护

### 动作效果
- 进行有效攻击后，获得治疗
